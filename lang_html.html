Le langage HTML

Définition

Le langage HTML, (HyperText Markup Language)c’est un langage de description de page Web, il permet de présenter le contenu Hypertext destiné à être affiché sur le navigateur. En d’autres termes, c’est un langage de codage qui sert à afficher des documents et des créations dans un navigateur (Google Chrome, Firefox, Edge, Safari, etc.)

Versions

On a HTML 1, HTML 2, HTML 3, HTML 4 et HTML 5 qui est la version récente.

On ne peut parler de Html sans évoquer le CSS (Cascading Style Sheets ) qui est un langage de style qui sert à soigner le design du contenu intégré en Html.
